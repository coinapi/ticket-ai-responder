The block number where the estimate was found.'feerate'
The estimated fee rate, denoted in BTC/kB.'errors'
A list of errors encountered during the processing, if any.'str'
A string describing the error in detail, if any error was encountered.NotesThe 'conf_target' parameter is essential to get an accurate fee estimate. It represents the number of blocks within which you want the transaction to be confirmed.The 'estimate_mode' parameter allows you to tailor the fee estimate according to your preference.
